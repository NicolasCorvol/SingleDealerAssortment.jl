SingleDealerAssortment.create_archetype_features — Method
create_archetype_features(
    instance::Instance,
    step_solution::Solution
) -> Matrix{Float64}

Create features per archetype. The first instance.nb_features columns correspond to static features. The last 6 columns correspond to dynamic features:

  • current stock and scaled with price
  • mean sales and scaled with price
  • mean days on lot and scaled with price
source
SingleDealerAssortment.create_stock_features — Method
create_stock_features(
    instance::Instance,
    step_solution::Solution,
    features_archetypes
) -> Any

Create features per stock level per archetype. The first instance.nb_features+6 columns correspond to static the archetype features. The last 8 columns correspond to dynamic stock features:

  • deviation from stock_inf and scaled with price
  • deviation from stock_sup and scaled with price
  • deviation from min_quota and i and scaled with price
  • deviation from mean stock and scaled with price
