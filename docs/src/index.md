# SingleDealerAssortment

Documentation for [SingleDealerAssortment](https://github.com/NicolasCorvol/SingleDealerAssortment.jl). This package is the first work of a CIFRE PhD in collaboration between Renault Group and CERMICS Laboratory. It explores new methods to solve the dynamic single dealer stock replenishment problem.  

If you have any feedback or question, feel free to [create an issue](https://github.com/NicolasCorvol/SingleDealerAssortment.jl/issues/new) or [contact me](mailto:nicolas.corvol@enpc.fr).
